# Yu-Gi-Oh! Duel Lite

Mini-aplicación en **Java Swing** que simula un duelo sencillo de Yu-Gi-Oh! entre un jugador y la máquina.  
Se obtienen cartas al azar desde la API [YGOProDeck](https://ygoprodeck.com/api-guide/).

## Objetivos
- Consumir una API REST desde Java (JSON).
- Aplicar POO con clases organizadas.
- Usar Swing para la interfaz de usuario.
- Manejar eventos con Listeners.
- Practicar trabajo en equipo con Git y GitHub.

## Requisitos técnicos
- Java 11+
- Maven o Gradle
- Librería ligera de JSON
